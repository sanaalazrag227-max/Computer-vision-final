# Détection d'objets avec YOLOv8

## Description

Ce projet utilise le modèle YOLOv8 Nano pour détecter automatiquement les objets présents dans une image.

## Technologies

- Python
- Ultralytics YOLOv8
- OpenCV

## Installation

```bash
pip install -r requirements.txt
```

## Exécution

```bash
python detect.py
```

## Résultat

Le résultat est enregistré automatiquement dans :

```
runs/detect/predict/
